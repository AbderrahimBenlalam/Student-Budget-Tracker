The Student Budget Tracker is a Java Swing-based desktop application designed to help students monitor their income and expenses efficiently. It provides an intuitive way to:

  - Record transactions (income & expenses)

  - Categorize spending (food, transport, rent, etc.)

  - Analyze finances with visual charts and monthly summaries

  - Export data to CSV for further analysis
